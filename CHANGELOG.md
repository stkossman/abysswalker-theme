# Change Log

All notable changes to the "abysswalker" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.0.1] - 2025-12-20

### Changed
- **UI Overhaul:** Removed the default blue focus borders in Explorer and Lists.
- **Accents:** Changed primary UI buttons (Source Control, Extensions) and list selections to use the theme's Crimson accent (`#bd7e7e`) instead of standard blue.
- **Readability:** Increased brightness of comments (`#606060`) and punctuation (`#757575`) to improve visibility on the deep dark background without breaking the muted atmosphere.
- **Tabs:** Active tab border is now colored to match the theme accents.

## [1.0.0] - 2025-12-18

### Added
- Initial release of Abysswalker Theme.
- Implemented "Void" background (`#1a1a1a`) for full immersion.
- Added minimalist syntax highlighting for Frontend stack (React, Astro, TypeScript, HTML, CSS).
- Added support for distinct component highlighting in JSX/TSX.
