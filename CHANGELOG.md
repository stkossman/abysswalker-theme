# Change Log

All notable changes to the "abysswalker" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.0.0] - 2025-12-18

### Added
- Initial release of Abysswalker Theme.
- Implemented "Void" background (`#1a1a1a`) for full immersion.
- Added minimalist syntax highlighting for Frontend stack (React, Astro, TypeScript, HTML, CSS).
- Added support for distinct component highlighting in JSX/TSX.
